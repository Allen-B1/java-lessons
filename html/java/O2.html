Classes: Member Fields

To make a custom data type, use classes.

class Main {
	// Custom Data Type!
	static class Person {
		public String name;
		public int age;
	}

	static void printInfo(Person p) {
		System.out.println(p.name + ": age " + p.age);
	}

	public static void main(String[] args) {
		Person p = new Person();
		p.name = "Hello";
		p.age = 31;
		printInfo(p);
	}
}
