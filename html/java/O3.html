Classes: Static Methods & Fields

Unlike normal methods & fields, static methods & fields are not associated with a specific instance of an object.

Main.java:

class Main {
	public static void main(String[] args) {
		Person p1 = new Person("Steve", 35);
		Person p2 = new Person("Bob", 5);
		p1.printInfo();

		// 2
		Person.printCounter();
	}
}

Person.java:

class Person {
	public String name;
	public int age;

	// A static field.
	// This is not tied to a specific instance.
	public static int counter = 0;

	public void printInfo() {
		System.out.println(this.name + ": age " + this.age);
	}

	public Person(String name, int age) {
		this.name = name;
		this.age = age;

		counter += 1;
	}

	public static void printCounter() {
		System.out.println(counter);

		// Static methods are not called
		// on a particular instance;
		// therefore, `this` is not
		// available. The following
		// would fail if uncommented.

		// System.out.println(this.name);
	}
}
