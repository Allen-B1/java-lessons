# Bitwise Operators
These aren't as commonly used, but they're listed here for the sake of completion in case you encounter them.

| Name | Expression |
|---|---|
| bitwise and | `a & b` |
| bitwise or | `a | b` |
| bitwise xor | `a ^ b` |
| bitwise not | `~a` |
| left bitshift | `a << b` |
| sign-extending right bitshift | `a >> b` |
| zero-extending right bitshift | `a >>> b` |
