## `switch` Statements
A `switch` statement lets you execute a piece of code depending on what value a specific expression is.

```java
switch (value) {
	case 1:
		// executed if value is 1
		break;
	case 2:
	case 3:
		// executed if value is 2 or 3
		break;
	default:
		// executed otherwise
}
```
