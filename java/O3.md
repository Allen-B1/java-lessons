# Classes: Methods Revisited
```java
class Main {
	static class Person {
		public String name;
		public int age;

		// Methods allow programmers
		// to define functionality
		// on custom types.
		//
		// `this` refers to the
		// current instance
		public void printInfo() {
			System.out.println(this.name + ": age " + this.age);
		}
	}

	public static void main(String[] args) {
		Person p = new Person();
		p.name = "Hello";
		p.age = 31;
		p.printInfo();
	}
}
```
