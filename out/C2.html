Loops

There are four kinds of loops: while, do...while, for, and for-each, which will be covered later.

while

while (condition) {
	// body
}

Executes body if condition is true. Repeat.

graph TD condition{condition} --> |true| body condition --> |false| next["next statement"] body --> condition

break statements

break;

can be placed inside of a while loop to end the loop early.

graph TD condition{condition} --> |true| body condition --> |false| next body --> condition body --> |break| next["next statement"]

continue statements

continue;

can be placed inside a while loop to skip the body and start over.

graph TD condition{condition} --> |true| body condition --> |false| next body --> condition body --> |break| next["next statement"] body --> |continue| condition

do...while

do {
	// body
} while (condition);

Executes body, then repeat if condition is true.

graph TD body --> condition condition{condition} --> |true| body condition --> |false| next body --> |break| condition body --> |continue| body

for

for (initialization; condition; postcondition) {
	// body
}

condition is a boolean expression. initialization and postcondition are statements.

graph TD init[initialization] --> cond{condition} cond --> |true| body body --> post[postcondition] post --> cond cond --> |false| next["next statement"] body --> |break| next body --> |continue| body
